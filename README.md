# ANDRA

A Next.js 14 + Tailwind CSS landing page for ANDRA, designed as a luxury personal strategy consulting one-page site and ready for deployment on Vercel.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Stack

- Next.js 14 App Router
- Tailwind CSS
- TypeScript
- Ready for Vercel deployment
